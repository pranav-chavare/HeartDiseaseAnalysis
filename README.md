# ❤️ Heart Disease Analysis

## 📌 Project Overview
Heart Disease Analysis is a data analysis project that helps identify patterns and risk factors related to heart disease using medical datasets.

## 🎯 Objectives
- Analyze heart disease dataset
- Identify important health indicators
- Perform data visualization
- Predict heart disease risk

## 🛠️ Technologies Used
- Tableu
- Python
- NumPy
- Jupyter Notebook

## 📂 Project Structure
1. Ideation Phase – Initial project idea and problem definition  
2. Requirement Analysis – Identifying data and tools required  
3. Project Design Phase – Designing workflow and architecture  
4. Project Planning Phase – Planning development steps  
5. Project Development Phase – Implementing the analysis  
6. Functional & Performance Testing – Validating results  
7. Documentation & Demo – Final project explanation

## 📊 Dataset
The dataset contains medical attributes such as:
- Age
- Cholesterol
- Blood Pressure
- Maximum Heart Rate
- Chest Pain Type

## 🚀 Expected Outcome
The project helps in understanding heart disease risk factors and provides insights through data analysis.
